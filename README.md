# YAPP 프로젝트 연습용 레포입니다.

## 시작하기
```
python3 -m venv myvenv
source myvenv/bin/activate
pip install -r requirements.txt
```

## 파일구조(변경여지 있음)
User - User, Feed, QuestList
Quest - Quest
Planet - Planet
Otherfunc - Trashcan, Token
